# Student Management System

A simple CRUD application built using Spring Boot, HTML, CSS, and JavaScript.

## Features

- Add Student
- View Students
- Delete Student
- REST APIs
- Responsive UI

## Tech Stack

- Java
- Spring Boot
- HTML
- CSS
- JavaScript
- Maven

## How to Run

1. Clone the repository.
2. Open it in VS Code or IntelliJ.
3. Run:

```bash
mvn spring-boot:run
```

4. Open:

```
http://localhost:8080
```
